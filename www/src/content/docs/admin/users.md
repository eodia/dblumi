---
title: User management
---

Admins can manage all users from **Settings → Administration → Users**.

## Actions

- **View** all registered users and their roles
- **Edit** a user's role (admin / editor / viewer)
- **Delete** a user and revoke their sessions

## Roles

| Role | Description |
|------|-------------|
| Admin | Full access to all features and administration |
| Editor | Can create connections and queries, manage their own data |
| Viewer | Read-only access to shared resources |

## Inviting users

Currently, users self-register. An admin must approve or assign roles after registration. Invite-only mode is on the roadmap.
