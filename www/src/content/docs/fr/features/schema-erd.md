Parcourez le schema de votre base de donnees et visualisez les relations entre tables.

![Vue d'ensemble avec explorateur de schema et diagramme ERD](/images/feature-schema.png)

## Explorateur de schema

La barre laterale affiche toutes les tables, vues et fonctions de votre base de donnees. Cliquez sur une table pour l'ouvrir dans l'editeur SQL.

## Explorateur de table

Cliquez sur **Parcourir** sur une table pour voir ses lignes, modifier des valeurs, inserer de nouvelles lignes ou supprimer des enregistrements (avec confirmation par glissement de securite).

## Editeur de structure

Modifiez votre schema directement depuis dblumi : ajoutez, modifiez ou supprimez des colonnes. Supporte les types de donnees specifiques a PostgreSQL, MySQL et Oracle.

## Diagramme ERD

La page Vue d'ensemble inclut un diagramme entite-relation de votre base de donnees. Il affiche toutes les tables et leurs colonnes d'un coup d'oeil.
