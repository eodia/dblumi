---
title: Requetes sauvegardees
---

Sauvegardez, organisez et partagez vos requetes SQL avec votre equipe.

## Sauvegarder une requete

Dans l'editeur SQL, cliquez sur **Sauvegarder** (ou `Ctrl+S`). Donnez un nom a votre requete et assignez-la optionnellement a un dossier.

## Dossiers

Organisez vos requetes dans des dossiers imbriques. Glissez-deposez pour reorganiser au sein d'un dossier.

## Epinglage

Epinglez les requetes frequemment utilisees depuis la page Vue d'ensemble. Les requetes epinglees apparaissent en haut de votre liste d'acces rapide.

## Partage

Les requetes peuvent etre partagees avec :
- **Des utilisateurs specifiques** — ils voient la requete dans leur liste
- **Des groupes** — tous les membres du groupe y ont acces

Les requetes partagees sont en lecture seule pour les destinataires, sauf si vous accordez l'acces en edition.

## Filtrage par connexion

Les requetes peuvent etre liees a une connexion specifique ou rester globales (visibles sur toutes les connexions).
