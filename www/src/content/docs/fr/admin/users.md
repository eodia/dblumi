---
title: Gestion des utilisateurs
---

Les administrateurs peuvent gerer tous les utilisateurs depuis **Parametres → Administration → Utilisateurs**.

## Actions

- **Voir** tous les utilisateurs inscrits et leurs roles
- **Modifier** le role d'un utilisateur (admin / editeur / lecteur)
- **Supprimer** un utilisateur et revoquer ses sessions

## Roles

| Role | Description |
|------|-------------|
| Admin | Acces complet a toutes les fonctionnalites et a l'administration |
| Editeur | Peut creer des connexions et requetes, gerer ses propres donnees |
| Lecteur | Acces en lecture seule aux ressources partagees |

## Inviter des utilisateurs

Actuellement, les utilisateurs s'inscrivent eux-memes. Un admin doit approuver ou assigner les roles apres l'inscription. Le mode invitation uniquement est prevu dans la roadmap.
